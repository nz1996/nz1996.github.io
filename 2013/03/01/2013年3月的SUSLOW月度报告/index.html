2013年3月的SUSLOW月度报告

发表于 2013-03-01 | 分类于 绝密

题目: 2013年3月的SUSLOW月度报告
发行日期: 2015-03-10
文件日期: 2013-03-01
描述: 1.此NSA文件从2013年3月开始提供有关GCSB活动的最新信息，包括使用针对东盟国家的WARRIOR PRIDE恶意软件和Kim Dotcom案例。2.美国国家安全局美国惠灵顿特别联络官的月度报告概述了FROSTBITE（BASILHAYDEN）的进展情况 - 一个渗透中国外交计算机网络的项目。
文件类型: 绝密

（U）3月亮点：

  • （U）情报更新：
    o（TS // SI // REL TO美国，FVEY）GCSB具有WARRIORPRIDE功能，可以针对东盟目标进行收集 。授权已过期; GCSB正在努力重建它。GCSB还致力从GCSB到NSA的数据传输机制。
    o（S // SI // REL TO USA，FVEY）为GCSB 3 - 5年HF策略提供输入。该战略概述了对Tangimoana站的建议改进，包括更换天线系统和将收集功能升级到宽带GLAIVE。该提案应尽快送交GCSB董事会作最终决定。
    o（U // FOUO）情报副主任，正在向内务部工作六个月 。他预计不会回到GCSB。是代理DDI到4月10日。尚未命名长期替代品。

  • （U）IA /网络更新：
    o（S // REL TO USA，FVEY）为响应新西兰境内有针对性的网络威胁的增加，GCSB确定了以下4个主要投资目标 ：

  • 使网络和系统不易受到影响网络利用

  • 改进对使用已知工具/技术的入侵的检测

  • 增加对新的和以前未知的工具/技术的发现

  • 在导致入侵之前积极破坏入侵来源
    o（U // FOUO）GCSB与社区黄金标准团队一起参加了VTC，讨论了使用CGS能力领域来支持更全面的网络安全和保障方法。GCSB概述了他们的风险管理/资产管理
    计划，CGS团队提供了CGS和可管理网络计划的背景知识。他们还提出将现有的GCSB缓解措施与NIST 800-53保持一致，然后针对CGS功能进行人行横道调整 。将安排后续职业训练局。
    o（C // REL）第23届密钥管理战略小组（KMSG）活动由GCSB于2013年3月19日至22日举办，每届FVEY代表出席伙伴。NSA团队通过VTC与当地的NSA IA联络人一起参加。

分类
o（U // FOUO）在DIA的恶意软件分析团队和FVEYs合作伙伴之间建立了信息共享链接 。针对与GCSB网络分析师共享的DIA报告的积极反馈 ，DIA同意将FVEYs合作伙伴添加到他们的分销中。
（U）问题：
（U // FOUO）新西兰政府正在审议三项立法提案，这将影响GCSB的运作：
1.（S // REL FVEY）2003年GCSB法案的更新，以扩大GCSG的核心职能，包括支持外部政府机构，并允许私营部门支持信息保障和网络安全。
2.（U // FOUO）加强了监察长和情报和安全委员会的监督。
3.（U // FOUO）新西兰政府组织的网络和供应链安全。

o（S // SI // REL TO USA，FVEY）GCSB 项目FROSTBITE / TAO项目 BASILHAYDEN：GCSB正在起草一份由GCSB主任签署的文件，其中概述了与TAO就此活动的互动。本文件旨在回答TAO关于允许其向前推进的操作的问题 。
