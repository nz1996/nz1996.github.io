（U）介绍
（U // FOUO）本手册将使旅行者熟悉准备他们根据OAKSTAR计划旅行的工具 。包括责任细节和关键支持组织的识别 。

（U）背景
（TS // SI // NF）前往OAKSTAR旅行主要涉及在美国大陆旅行。以下每个子弹的名字是指SSO项目名称; 第二个名称是NSA/CSS商业解决方案中心的名称
（NCSC）通过标识合作伙伴关系。SSO和NCSC与以下每个合作伙伴共同管理这些计划。
（TS // SI // NF）这些计划涉及与美国公司合作。虽然旅行主要是U // FOUO，但所遇到的人员和公司的身份 ，设施地址等都受到SSO WHIPGENIE ECI和NCSC AAA ECI的保护。因此，您必须持有两个ECl中的一个在该计划内旅行。如果您目前不持有EC1且需要参加该计划，则必须在旅行日期之前先与计划主任（PD），副计划主任（DPD）或任务计划经理（MPM）联系 。

（U）一般管理局
（U）本地或美国大陆
（C）特殊来源行动（SSO），计划管理司（PMD）协调与合作伙伴，批准为TDY提供资金，处理与NCSC的互动 ，并在适当时帮助通过许可。这包括与SSO工程技术服务（ETS）和运营和发现部门（ODD）代表以及其他政府机构合作完成此任务。
（U）CONUS
（U // FOUO）每次旅行都应该有指定的团队领导。团队负责人负责部署的技术方面。这包括及时向NBP320的OAKSTAR PMD团队提供更新。通常，如果在TDY上，PMD成员将担任此角色。被指定为访问负责人的人员必须在访问结束后的五天内提交旅行报告。如果需要，请联系PMD获取模板和/或之前访问的样本。
（U）旅行
（U // FOUO）通常不允许承包商乘坐这些计划或与合作伙伴会面 。如果必须例外，承包商需要提前通知PD或DPD，并与NCSC进行协调。
（C）PD，DPD或MPM将负责安排与合作伙伴的所有会议，有时与NCSC PD一起。可根据需要将安排委托给ODD / ETS代表。
（U）旅行计划和规划信息
（U // FOUO）LL42公司旅行服务（在NSA内联网上“旅行”）可以协助公务旅行的各个方面，包括政府旅行收费卡的信息，通过^^预订^^^^^ | 官方政府护照，请求旅行授权（RTA）表格，旅行清单等等。还将安排租车和酒店。与所有旅行一样，请务必检查租车和实际费用。
（TS // NF）这是一些有用的旅行信息。
