智力机会评估 - Juniper

发表于 2011-02-03 | 分类于 绝密

题目: 智力机会评估 - Juniper
发行日期: 2015-12-23
文件日期: 2011-02-03
描述: 此GCHQ报告日期为2011年2月3日，由借调的NSA工作人员撰写，暗示了代理商对Juniper网络公司生产的13种防火墙模型的能力。
文件类型: 绝密
关键字: Juniper（瞻博网络）;GCHQ;

背景

  • Juniper Networks，Inc。总部位于美国加利福尼亚州桑尼维尔市，是一家高性能互联网协议网络产品公司。瞻博网络的主要产品包括T系列，Mi系列，E系列，MX系列和J系列路由器系列，EX系列以太网交换机以及NetScreen和SRX系列安全产品。
  • 虽然瞻博网络不一定是任何一个领域的市场份额领导者，但它们是SIGINT在几个重要市场上的强大竞争对手和技术领导者透视。凭借多年来提供最高密度的路由器，瞻博网络在许多国家/地区处于互联网核心。
  • 瞻博网络在核心IP路由以及企业网络防火墙和SSL VPN市场中的领先地位意味着SIGINT社区应该跟上瞻博网络技术的步伐，以便随着时间的推移保持CNE访问。

目前的漏洞利用功能

  • Juniper NetScreen防火墙型号为NS5gt，N25，NS50，NS500，NS204，NS208，NS5200，NS5000，SSG5，SSG20，SSG140，ISG 1000，ISG 2000.根据固件版本的不同，可能需要进行一些逆向工程 。
  • Juniper路由器：M320目前正在开发中，我们预计到2010年底将得到全面支持。

建议和预期成果

  • 在硬目标国家的网络上发现Juniper设备，以评估现有功能对Juniper的潜在利用机会。
  • 如果可以利用其他设备模型，评估潜在的额外可定位网络 （例如，如果我们可以利用MX系列路由器，则可以利用网络X，Y和Z）。
  • 评估开发JUNOS操作系统的工作。

瞻博网络概述
瞻博网络公司历史
Juniper网络公司（纽约证券交易所股票代码：JNPR）总部位于美国加利福尼亚州桑尼维尔市，是一家成立于1996年的高性能互联网协议网络产品公司。瞻博网络的主要产品包括T系列，M系列，E系列，MX系列和J系列路由器系列，EX系列以太网交换机以及NetScreen和SRX系列安全产品。瞻博网络的JUNOS网络操作系统可在大多数瞻博网络产品上运行。
1995年，施乐公司帕洛阿尔托研究中心的首席科学家Pradeep Sindhu从假期回来， 想要创办一家公司，提供高性能路由器，以支持快速兴起的互联网。Sindhu于1996年2月以20万美元的种子开办了该公司钱。他从Sun Microsystems聘请了工程师Bjorn Liencres，从MCI 招募了Dennis Ferguson 。对于商业专业知识，Sindhu招募了StrataCom的联合创始人Scott Kriens.Juniper认为技术发展具有战略优势，尽管2008年经济不景气，但还是 决定不削减8亿美元的研发费用预算。瞻博网络于1998年9月推出了第一款产品M40路由器。该产品是第一款产品实现数据包转发，可以在半机架外形中维持8个OC48c端口的线速数据包转发 。这是一项关键的技术改进，允许无限制的互联网增长，并使瞻博网络在以前由思科系统主导的市场中占据一席之地 。瞻博网络通过提供M160实现OC-192转发率来维持市场发展势头 。到2000年，Juniper占据了互联网核心路由器市场的30％。2002年，Juniper宣布T640具有40千兆/插槽性能和太比特级系统扩展能力。2002年，瞻博网络还宣布计划将其互联网核心路由器产品线扩展到边缘，并开始确定其他市场，如企业路由/交换和安全性。通过2002年底，瞻博网络已经通过瞻博网络E系列渗透到宽带聚合领域 。通过M40e（2002）和M7i-M10i（2003）系统将Juniper M系列技术扩展到网络边缘，进一步支持了这一趋势。接下来，瞻博网络通过从NetScreen Technologies获得的技术以及内部开发的低端J系列路由器系列进入企业和安全领域。2006年，瞻博网络推出了高度集成的边缘专用10 Gbit / s芯片组（I-Chip），构成了高度冗余的M120边缘路由器和瞻博网络MX系列以太网专用载波路由器的基础。由于以太网服务日益增长的重要性，MX系列获得了超额收益在不到18个月的时间内完成250个帐户，并将其硬件借给SRX系列安全设备。2009款MX每插槽可提供高达120Gbps（全双工）的功能。在2007年的核心业务上，瞻博网络发布了一款100Gbps /插槽的Juniper T1600路由器。该T1600是市售去最密集的核心路由器到2010年的第一个产品，提供商业实现100GE接口（802.3ba标准）的。
JUNOS是内部操作系统，可在Juniper的大多数网络设备上运行，包括路由，交换和安全平台。JUNOS是第一款商用的全功能模块化操作系统，具有完整的内存保护功能，可用于路由产品。JUNOS与Cisco IOS-XR和Alcatel-Lucent SR-OS等其他模块化系统竞争。JUNOS具有垂直和水平模块性，并为称为“JUNOS Space” 的第三方应用程序提供API 。尽管JUNOS最初源自FreeBSD，但随后的产品开发导致了主要的内核和基础架构改进，如In-Service Software Upgrade和实时数据包转发平面。

当前产品系列

  • E系列路由器是宽带边缘路由器。E系列由瞻博网络于2002年收购的Unisphere开发 .E系列路由器运行继承自Unisphere 的JUNOS操作系统 。J，M，T和MX系列路由器运行JUNOS。
  • J系列路由器是小型客户端设备或企业路由器。
  • M系列路由器是多业务边缘路由器。
  • T系列路由器是大型核心路由器。
  • MX系列路由器是以太网服务路由器。
    EX系列交换机 - 瞻博网络的交换机产品于2008年推出并运行JUNOS。提供具有完整或部分PoE功能的固定和模块化外形，EX代表Juniper的出价适用于企业和成本优化的以太网市场，增强“一体操作系统”战略，并在2009年第四季度创造7400万美元的收入。SRXSeries动态服务网关是运行JUNOS的一系列安全服务设备。从分支机构型号到世界上最快的防火墙SRX 5800。在一个机箱中结合 安全性，路由和交换。安全功能包括以前在ScreenOS上找到的完整UTM功能 ，包括Web过滤，IDP和防病毒。
    NetScreen SSG系列和ISG系列防火墙运行ScreenOS操作系统，并提供防火墙，防病毒，入侵防护和VPN服务。获得NetScreen技术，他们运行ScreenOS而不是JUNOS。这些目标是中小型企业。ISG系列具有更高级的IDP和虚拟化功能以及更高的性能。
    Secure Access产品为远程用户提供基于SSL的VPN服务，无需专门的客户端。NSM网络和安全管理器是瞻博网络设备的企业级管理工具 ，具有对多个瞻博网络设备的单点堡垒控制，系统日志主机和配置备份存储库以及提供分布式分层功能的NSMXpress设备 。

瞻博网络作为目标
尽管瞻博网络不一定是任何一个领域的市场份额领导者，但从SIGINT的角度来看，它们是几个重要市场的强大 竞争对手和技术领导者。凭借多年来提供最高密度的路由器，瞻博网络在许多国家/地区处于互联网核心 。随着电信服务提供商转向所有IP网络，瞻博网络将在融合网络中发挥越来越重要的作用。瞻博网络已经证明自己很擅长利用其高密度服务器技术挑战边缘服务器和企业网络防火墙市场的市场领导者 。在另一个利基市场，但对SIGINT非常重要的市场 ，瞻博网络被视为销售SSL VPN技术最有竞争力的竞争对手。

  • 利用高密度路由器在运营商空间中建立良好的地位
  • 可靠的竞争替代思科核心路由的主导地位
  • 运营商以太网在数量和范围上不断增长
  • IP流量增长持续不断

瞻博网络作为威胁
瞻博网络在核心IP路由以及企业网络防火墙和SSL VPN市场中的领导地位，意味着SIGINT社区应该跟上瞻博网络技术，以便维护CNE随着时间的推移。威胁来自瞻博网络的投资，并强调
成为安全领导者。如果SIGINT社区落后，如果Juniper继续快速提高其安全性，则可能需要数年才能重新获得 Juniper防火墙或路由器访问功能。

瞻博网络
全球IP网络的目标使用 - 瞻博网络核心路由器可以在整个互联网和其他所有网络中找到高容量IP网络。例子太多，无法引用。以FLAG Telecom案例研究为例。
巴基斯坦 - 瞻博网络防火墙是针对巴基斯坦政府/军事安全网络的高优先级HEADRESS NU项目的核心 。虽然巴基斯坦的核心互联网路由器都是思科，但Juniper通常被视为连接到核心网络的边缘路由器。Juniper路由器部署在Mobilink网络中，可能还部署在Telenor中。
阿富汗 - 没有瞻博网络存在的证据
CT经纪人 - 尽管瞻博网络已多次被提及与经纪人目标有关，但唯一的证据是瞻博网络路由器被用于小规模的审判，没有进一步采取。
CT也门 - 瞻博网络为也门电信提供安全硬件，并为TeleYemen与Verizon的VoIP连接提供防火墙 。
CT沙特阿拉伯 -
中国 - 瞻博网络通过位于香港的瞻博网络中国有限公司在中国拥有强大的业务 。以江西电子政府案例研究为例。下面列出了新闻稿 ，作为Juniper在中国的另一个例子。
瞻博网络路由平台构成中国下一代互联网。T系列核心平台和M系列多业务路由器的核心，为全球最大的IPv6网络 提供基础设施2005年11月30日，加利福尼亚州桑尼维尔 - 瞻博网络公司（纳斯达克股票代码：JNPR），今天宣布其M系列和T系列路由平台已被选为中国下一代互联网（CNGI）项目的核心。CNGI项目是中国政府资助的一项在中国推广互联网协议第6版（IPv6）的计划，预计将成为世界上最的IPv6网络。瞻博网络平台的选择是凭借其行业领先的业界领先的IPv6功能，并将部署在CNGI的参与网络中，包括中国教育和研究网络（CERNET2），中国移动，中国网通，中国铁通，中国电信和中国联通。

利用Juniper的当前和计划工作
GCHQ目前具有以下漏洞攻击能力：

  • Juniper NetScreen防火墙型号NS5gt，N25，NS50，NS500，NS204，NS208，NS5200，NS5000，SSG5，SSG20 ，SSG140，ISG 1000，ISG 2000.根据固件版本的不同，可能需要进行一些逆向工程 。
  • Juniper路由器：M320目前正在开发中，我们希望在2010年底之前获得全面支持 。
  • 目前尚不支持其他型号。
  • 在2010年CY20期间，与NSA共享的瞻博网络技术得到了显着改善，以利用GCHQ具有首要访问权限的多个目标网络。

评估潜在机会
多年来，利用瞻博网络服务器和防火墙的能力将带来很多好处。如今，瞻博网络已成为互联网上的主要硬件提供商。随着融合IP网络和瞻博网络技术领先地位的增长， 我们计Juniper
将在未来几年内实现显着增长。当买家寻找同时也是技术领导者的替代供应商时，瞻博网络是思科的强大竞争对手 。当买家寻求成本较低的替代供应商时，华为是同一领域的另一个竞争对手 。瞻博网络作为一家美国公司，具有潜在的机会和复杂性。如果NSA存在公司关系，则有可能利用公司关系。任何GCHQ利用瞻博网络的努力都必须从与NSA的密切协调开始。
Juniper系列产品在使用JUNOS操作系统时有些同质。这可能会为利用JUNOS中的安全漏洞创造机会，并将其推广到更广泛的Juniper路由产品线。

建议和预期成果
2.利用当今可用的内容（EWAT）：捕获现有机会

  • 记录利用瞻博网络设备的当前功能。
  • 在硬目标国家的网络上发现Juniper设备（潜在的Juniper开发机会）。
  • 验证潜在的Juniper利用机会1）适应当前的Juniper漏洞利用功能2）针对潜在的智能收益进行目标中心评估。
  • 对经过验证的Juniper利用机会和业务决策的影响评估 。

3.展开瞻博网络漏洞利用功能：创造未来机遇

  • 在硬目标国家的网络上发现瞻博网络设备（潜在的瞻博网络开发机会）。
  • 如果可以使用其他设备型号，则评估可能的其他可定位网络利用（例如，如果我们可以利用MX系列路由器，那么可以利用网络X，Y和Z）。
  • 如果可以利用其他网络，评估潜在的情报利益。
  • 创建其他Juniper模型漏洞的影响评估。

4.JUNOS剥削：

  • 绝大多数当前的Juniper漏洞都是针对运行ScreenOS操作系统的防火墙 。
  • Juniper将随着时间的推移将所有产品迁移到JUNOS操作系统。
  • 确保JUNOS开发能力的努力应该对广泛的Juniper产品产生影响。
